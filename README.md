# Python Bug Fix Agent

## Description
A Python-based bug fix agent that automatically identifies and fixes bugs in Python code using AI-powered analysis and code refactoring techniques.

## Features
- Automatic bug detection and analysis
- Intelligent code suggestions for fixes
- Code cleanup and optimization
- Support for multiple Python files

## Installation
To install the required dependencies, run:
```bash
pip install -r requirements.txt
```

## Usage
To run the bug fix agent:
```bash
python main.py
```

## Files
- `main.py` - Main entry point for the application
- `auto_clean.py` - Automatic code cleanup utilities

## Requirements
Python 3.7 or higher
